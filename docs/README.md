# AI-Remote CTO System

This is the AI-Remote CTO System, designed to provide multi-repository AI engineering control, task generation, multi-agent validation, learning from CI/CD results, cross-project intelligence, and architectural evolution suggestions.

## Setup

1.  **Clone this repository.**
2.  **Install dependencies:** `npm install`
3.  **Configure environment variables** in a `.env` file (see `.env.example`).
4.  **Run the Telegram bot:** `node src/bot/telegramBot.js`

## Features

*   Telegram-based task generation
*   Multi-agent validation (Planner, Reviewer, Security, DevOps)
*   Dependency graphing and impact analysis
*   Cross-project learning and architectural evolution proposals
*   Human-in-the-loop approval for all actions
